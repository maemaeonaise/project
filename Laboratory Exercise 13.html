Computer Hardware

Name Functions
Mouse The computer mouse is a hand-held pointing device used to manipulate objects on a computer screen.
Keyboard It is a typewriter-style device that enable users to interact with their computers.
Monitor It is a hardware device that displays the video and graphics information generated by the computer.
Printer The output devices that generate hard copies of electronic data stored on the conputer usually in the form of text or images on paper.
CPU It is responsible for processing all information from programs run by your computer.
Motherboard It is the main printed circuit board of a computer.It allocates power to other components.
Hard Disk Drive It is the main data storage device of a computer.This is where operating system,software titles are stored.
Speaker It is an output device used to listening to any sound from the computer.
RAM It is the physical hardware inside a computer that temporarily stores data.
Joystick Is an input device that used by a player to interactt with a computer game.
